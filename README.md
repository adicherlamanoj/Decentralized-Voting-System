# 🗳️ Decentralized Voting System

A blockchain-based voting application that ensures **transparency, security, and tamper-proof elections** using smart contracts.

---

## 🚀 Features

* 🧾 Transparent voting process
* 🔐 Tamper-proof system using blockchain
* 👤 One person, one vote
* 📊 Real-time vote counting
* 🌐 Decentralized (no central authority)

---

## 🛠️ Tech Stack

* **Blockchain:** Ethereum
* **Smart Contracts:** Solidity
* **Framework:** Truffle
* **Local Blockchain:** Ganache
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js

---

## 📁 Project Structure

```
contracts/        → Smart contracts  
migrations/       → Deployment scripts  
public/           → Frontend files  
Database_API/     → Backend API  
```

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/Decentralized-Voting-System.git
cd Decentralized-Voting-System
```

### 2️⃣ Install dependencies

```
npm install
```

### 3️⃣ Start Ganache

* Open Ganache and run local blockchain

### 4️⃣ Compile smart contracts

```
truffle compile
```

### 5️⃣ Deploy contracts

```
truffle migrate
```

### 6️⃣ Run the application

```
npm start
```
